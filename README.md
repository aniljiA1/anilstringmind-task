Project Overview

This platform allows startups to showcase benefits to employees or users.

Backend handles API endpoints for benefits, authentication, and data storage.

Frontend provides a modern, responsive UI with animations and smooth interactions.

Features

Fetch and display benefits dynamically

Claim benefits (backend API support)

Mobile-friendly, responsive design

Smooth animations using Framer Motion

TypeScript + strict type checking

Easy deployment on Vercel (frontend) and Render (backend)

Tech Stack
Layer	Technology
Frontend	Next.js 16, TypeScript, React, Tailwind CSS, Framer Motion
Backend	Node.js, Express, TypeScript, MongoDB / Your DB
Deployment	Vercel (frontend), Render (backend)
Others	CORS, dotenv, ESLint

Backend Setup

Running Locally
Backend

npm run dev   # or npm start

Frontend
cd frontend
npm run dev

Backend runs on: http://localhost:5000
